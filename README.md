## Running local

Clone the project

```bash
  git clone https://github.com/Gabriel28/warmup1-fiap-crud.git
```

Enter the directory project

```bash
  cd my-project
```

Install dependecies

```bash
  npm install
```

Start server

```bash
  npm run dev
```


## Stack

**Back-end:** Node, Express
